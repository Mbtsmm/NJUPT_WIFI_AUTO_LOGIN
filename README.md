# NJUPT_WIFI_AUTO_LOGIN
## 南京邮电大学WiFi自动登录程序
## 请在"登录信息.txt"中输入登录信息
## 第一行为账号
## 第二行为密码
## 第三行为网络类型（NJUPT输入1，NJUPT-CMCC输入2，NJUPT-CHINANET输入3）
## 打开main.exe即可登录WiFi
## 可设置开机自启动或连接WiFi后启动，自动登录WiFi
